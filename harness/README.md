Note: This is work in progress.

Usage:
1. Update hq/scripts/setenv.sh with your ACCOUNT_ID and API_KEY. 
2. Update HQ_HOME to the location where hq is cloned
3. source setenv.sh 


How to Test

$ hq 

Usage: hq <module/query.hql> VAR1 VAR2 VAR3...
